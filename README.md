# Pokedex

A REPL pokedex on the command line in Typescript. You explore locations, catch and inspect various pokemons and many more.

## Run the project

- `pnpm install` - To install required packages
- `pnpm run dev` - To run the code in dev mode

## Some Commands

- `catch <pokemon_name>`: Attempt to catch a pokemon
- `exit`: Exit the Pokedex
- `explore <location_name>`: Explore a location
- `help`: Displays a help message
- `inspect <pokemon_name>`: View details about a caught pokemon
- `map`: Get the next page of locations
- `mapb`: Get the previous page of locations
- `pokedex`: See all the pokemon you've caught
